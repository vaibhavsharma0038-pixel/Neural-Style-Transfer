# Neural-Style-Transfer

This project implements Neural Style Transfer (NST) using a Style-Attentional Network (SANet) to combine the content of one image with the artistic style of another.

# Features
- Attention-based Neural Style Transfer
- VGG19 feature extraction
- SANet attention mechanism
- High-quality stylized outputs
- Gradio web interface
- PSNR & SSIM evaluation metrics

# Technologies Used
- Python
- PyTorch
- Torchvision
- Gradio
- Matplotlib

# Model Architecture
1. VGG19 Encoder  
2. SANet Attention Module  
3. Decoder Network
